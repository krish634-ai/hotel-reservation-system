# StayHub

> **A Premium Hotel Reservation Platform with a Modern Booking Experience**

StayHub is a full-stack hotel reservation platform built to deliver a seamless booking experience through elegant UI design, smooth animations, and a responsive user interface. The platform enables users to discover premium hotels, explore detailed property information, and make reservations with ease while providing hotel owners with tools to manage their listings.

Designed with a focus on aesthetics and performance, StayHub combines modern web technologies with a cloud-based database to create a scalable and intuitive booking platform.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/stayhub.git
cd stayhub
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file:

```env
MONGODB_URI=your_mongodb_connection_string
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
